# Экспертная система «Мистер Принтер» 🖨️

Программное решение для автоматизации подбора периферийного оборудования на основе пользовательских требований. (2024 г.).

## 🧠 Описание проекта
Система представляет собой интеллектуального помощника, который преобразует абстрактные пожелания пользователя (например, «нужен для офиса», «хочу дешевле») в конкретные технические параметры (PPM, DPI, цена) с помощью **продукционной модели знаний**.

### Ключевые особенности:
* **Логический вывод:** Реализовано дерево решений, включающее 24 уникальных правила подбора.
* **ORM-архитектура:** Работа с данными организована через SQLAlchemy, что позволяет легко сменить базу данных (например, с SQLite на PostgreSQL).
* **Гибкая фильтрация:** Реализован динамический сборщик фильтров (mapping), сопоставляющий коды решений с параметрами БД.
* **Desktop UI:** Графический интерфейс на PySide6 (Qt) с адаптивной версткой элементов.

## 🛠 Технологический стек
* **Backend:** Python 3.12, SQLAlchemy (ORM)
* **GUI:** PySide6 (Qt for Python)
* **Database:** SQLite
* **Deployment:** PyInstaller (сборка в автономный `.exe`)

## 📐 Архитектура
Проект построен по модульному принципу:
1. `engine.py` — **Ядро:** обработка событий интерфейса и формирование запросов к БД.
2. `map.py` — **База знаний:** описание логики дерева решений и мэппинг технических характеристик.
3. `ui.py` — **Интерфейс:** код, сгенерированный из Qt Designer.
4. `mrPrinter.db` — **Тестовая база данных:** реляционные таблицы с характеристиками печатающих устройств.
